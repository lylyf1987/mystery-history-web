# Technology Stack

## Core Platform
- **Static Site Generator**: Hugo (fast, simple, great for content sites)
- **Hosting**: Netlify (free tier, CI/CD included)
- **Domain**: Cloudflare or Namecheap (~$10-15/year)

## Content Management
- **Writing**: Markdown files
- **Images**: Optimized static assets
- **Version Control**: Git + GitHub

## Analytics & SEO
- **Analytics**: Google Analytics 4 (free)
- **Search Console**: Google Search Console (free)
- **SEO**: Built-in Hugo SEO features

## Monetization (Future)
- **Ads**: Google AdSense
- **Newsletter**: Buttondown (free tier)
- **Donations**: Ko-fi or Buy Me a Coffee
