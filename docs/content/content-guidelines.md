# Content Guidelines

## Content Criteria
1. **Mystery/Unknown**: Stories not widely known
2. **Trustable**: Must have reliable sources
3. **Engaging**: Well-written, compelling narrative

## Source Requirements
- Primary sources preferred (academic papers, archives)
- Secondary sources: Reputable publications, documentaries
- Always cite references
- Fact-check before publishing

## Daily Update Process
1. Research topic (30-60 min)
2. Verify sources (15-30 min)
3. Write content (60-90 min)
4. Review & publish (15 min)

## Content Categories
- Historical Mysteries
- Unsolved Events
- Lesser-known Historical Figures
- Archaeological Discoveries
- Ancient Civilizations

## Image Requirements (CRITICAL)

### Category Bar Images
- **MUST be directly related** to the post topic or title
- **NO generic placeholder images** - images must specifically represent the subject
- Use primary sources when available (Wikimedia Commons, museum collections, official sources)
- Verify image relevance before using:
  - Does it show the actual artifact/location/event described?
  - Is it historically or contextually accurate?
  - Will visitors understand the connection immediately?

### Image Quality Standards
- Resolution: Minimum 400px width for category bars
- Source: Prefer Wikimedia Commons (public domain/CC licensed), reputable news sources, or museum collections
- Alt text: Must accurately describe the image and its subject
- Test: Image link must be valid (no broken URLs)

### Before Publishing Checklist
- [ ] Image is directly related to content (not generic)
- [ ] Image URL is valid and loads correctly
- [ ] Alt text is descriptive and accurate
- [ ] Source is reputable and properly licensed
