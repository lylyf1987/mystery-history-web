# Mystery History - Article Style Guide

## Voice & Tone
- **Voice**: Curious, knowledgeable, slightly mysterious but credible
- **Tone**: Engaging storytelling with factual foundation
- **Perspective**: Third-person narrative

## Article Structure

### 1. Title (H1)
- Compelling, includes mystery element
- 50-60 characters for SEO
- Example: "The Lost City of Heracleion: Egypt's Sunken Treasure"

### 2. Opening Hook (2-3 sentences)
- Start with intrigue or a surprising fact
- Set the scene and time period
- Draw reader in immediately

### 3. Background/Context (1-2 paragraphs)
- Historical setting
- Key figures or locations
- Why this matters

### 4. The Mystery Unfolds (3-4 paragraphs)
- Main narrative body
- Key discoveries or events
- Conflicting theories or perspectives

### 5. Theories & Explanations (2-3 paragraphs)
- Leading theories (scientific, historical, alternative)
- Evidence for and against each
- Maintain balanced perspective

### 6. Current Status (1 paragraph)
- What's happening now
- Ongoing research or investigations
- Open questions remaining

### 7. Sources Section
- List 3-5 credible sources
- Include books, academic papers, reputable websites
- Format: [Title](URL) - Brief description

## Formatting Guidelines

### Paragraphs
- 3-5 sentences maximum
- Use white space for readability
- Break up long sections with subheadings

### Subheadings (H2, H3)
- Clear and descriptive
- Use for major section breaks
- Example: "## The Discovery", "### Leading Theories"

### Emphasis
- Use **bold** for key terms or names
- Use *italics* for book titles, ship names, foreign words
- Avoid overuse

### Quotes
- Use blockquotes for significant quotes
- Attribute with source

## Word Count Guidelines
- **Minimum**: 800 words
- **Target**: 1,200-1,500 words
- **Maximum**: 2,500 words (split into series if longer)

## SEO Best Practices
- Include primary keyword in title and first paragraph
- Use related keywords naturally throughout
- Write meta description (150-160 characters)
- Include internal links to related articles
- Use descriptive image alt text

## Image Guidelines
- Use high-quality, atmospheric images
- Source from Unsplash, Pexels, or public domain
- Include captions where relevant
- Optimize file sizes for web

## Fact-Checking Standards
- Verify all dates, names, and locations
- Cross-reference multiple sources
- Note when information is disputed
- Avoid sensationalism over accuracy
